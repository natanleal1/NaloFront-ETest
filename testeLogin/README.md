## Login Test - English

![](assets/UserInterface.png)

A responsive authentication system with Login, Sign Up, and Home pages built with vanilla JavaScript and modern CSS. Features form validation, loading states, and interactive navigation.

**Key Features:**
- Home page with navigation bar (Bootstrap)
- Login and Sign Up forms with client-side handling
- Form submission with loading feedback
- Responsive design for all devices
- Gradient backgrounds and smooth transitions

**Technologies:** HTML5, CSS3, JavaScript, Bootstrap 5

**Author:** Natan Leal (Nalo)

## Teste Login - Português

![](assets/UserInterface.png)

Um sistema responsivo de autenticação com páginas de Login, Cadastro e Home construído com JavaScript vanilla e CSS moderno. Inclui validação de formulários, estados de carregamento e navegação interativa.

**Funcionalidades Principais:**
- Página inicial com barra de navegação (Bootstrap)
- Formulários de Login e Cadastro com tratamento no cliente
- Submissão de formulário com feedback de carregamento
- Design responsivo para todos os dispositivos
- Fundos gradiente e transições suaves

**Tecnologias:** HTML5, CSS3, JavaScript, Bootstrap 5

**Autor:** Natan Leal (Nalo)

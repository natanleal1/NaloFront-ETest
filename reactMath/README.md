## ReactMath - English

A modern React + Vite application for financial calculations featuring interactive interest rate computations. Built with React hooks for state management and component-based architecture.

![](src/assets/UserInterface.png)

**Key Features:**
- Simple Interest Calculator: Compute `I = P × r × t`
- Compound Interest Calculator: Calculate exponential growth with periodic compounding
- Present Value Calculator: Determine current value of future cash flows
- Dynamic tab navigation with React state management
- Real-time calculation results
- Clean, responsive user interface

**Technologies:** React 19, Vite 7, JavaScript ES6+, CSS3

**Author:** Natan Leal (Nalo)

## ReactMath - Português

Uma aplicação moderna React + Vite para cálculos financeiros com computações interativas de taxas de juros. Construída com React hooks para gerenciamento de estado e arquitetura baseada em componentes.

![](src/assets/UserInterface.png)

**Funcionalidades Principais:**
- Calculadora de Juros Simples: Calcule usando a fórmula `I = P × r × t`
- Calculadora de Juros Compostos: Calcule crescimento exponencial com capitalização periódica
- Calculadora de Valor Presente: Determine o valor presente de fluxos de caixa futuros
- Navegação dinâmica por abas com gerenciamento de estado React
- Resultados de cálculo em tempo real
- Interface limpa e responsiva

**Tecnologias:** React 19, Vite 7, JavaScript ES6+, CSS3

**Autor:** Natan Leal (Nalo)

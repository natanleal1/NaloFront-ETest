# ReactMath - Uma aplicação para calculos de Juros

Nesta Aplicação através das opções listadas é possível realizar calculos de juros simples e compostos, além de calculo de valor presente. Através dessa aplicação conceitos basilares de `React`+`Vite` foram aplicados.
